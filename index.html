BlockCoin Documentation

Welcome to the official documentation for the BlockCoin currency system for Scratch. This system uses Scratchattach to connect shops and games to a centralized server and then uses a captcha to verify transactions safely and easily.

How to Add BlockCoin to Your Project

  1. Visit the official BlockCoin project at scratch.mit.edu/projects/1331635541/.
  2. Open your Backpack at the bottom of the screen.
  3. Backpack the "Cloud Requests" sprite and add it into your project.
  4. Make sure to credit @Timmccool in your project description or notes and credits.

How to Get Approval to Use BlockCoin in Your Project

Visit the official BlockCoin forum page (link) and fill out the application. It should take 24-48 hours to get approval.

In the meantime, the server will not process any requests from your project.

How to Use BlockCoin in Your Project

To perform transactions using BlockCoin, use the PayCoins (amount) custom block.

This block will poll the Python server for a captcha that will require the user to comment a verification code on the official project (scratch.mit.edu/projects/1331635541/).

BlockCoin Commands

These are the core commands you can run using the blocks inside the Cloud Requests sprite:

Command Name What It Does
get_captcha Returns a comment verification captcha to your project.
get_owner Finds the registered owner of the project on the server (all transactions will be paid to the owner).
transaction Pays an amount to the project owner.

Official Script Layouts

Recreate these exact blocks inside your project to handle payments seamlessly.

The Official Payment Block (PayCoins Definition)

define PayCoins (Amount)
Send Request | request name: [get_captcha] argument1: (username) and wait
ask [response] and wait
Send request | request name: [get_owner] and wait
ask (join (join (join [Do you authorize a transaction of ] (Amount)) [ to ]) (join (response) [ (y/n)])) and wait
if <(answer) = [y]> then
  Send request | request name: [transaction] argument1: (Amount) argument2: (username) and wait
end

Safety Features

BlockCoin contains built-in safety features to preserve the integrity of projects on the server and the users using them:

  • It uses Scratchattach’s built-in verification system.
  • It features anti-bot and spam limiters on the server side.
