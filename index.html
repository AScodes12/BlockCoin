BlockCoin Documentation

Welcome to the official documentation for the BlockCoin currency system for Scratch. This system uses Scratchattach to connect shops and games to a centralized server and then uses a captcha to verify transactions safely and easily.

How to Add BlockCoin to Your Game

  1. Open the official project editor at https://scratch.mit.edu/projects/1331635541/editor/.
  2. Open your Backpack at the bottom of the screen.
  3. Drag the sprite named "Cloud Requests" into your backpack, then open your own project and drag it in.
  4. Important: You must give credit to @timmccool in your project's Notes and Credits for the cloud request system!

Scratch Command Reference

These are the core commands you can run using the blocks inside the Cloud Requests sprite:

Command Name What to Put in Argument 1 What It Does
get_captcha Your username variable Asks the server for your unique transaction code.
get_owner (Leave blank) Finds the username of the person who owns the project.
transaction The amount of coins to pay Takes coins out of the user's account and completes the payment.

Official Script Layouts

Recreate these exact blocks inside your project to make payments work seamlessly.

1. The Custom Payment Block (PayCoins Definition)

define PayCoins (Amount)
Send Request | request name: [get_captcha] argument1: (username) and wait
ask [response] and wait
Send request | request name: [get_owner] and wait
ask (join [Do you authorize a transaction of ] (join (Amount) (join [ to ] (join (response) [ (y/n)])))) and wait
if <(answer) = [y]> then
  Send request | request name: [transaction] argument1: (Amount) argument2: (username) and wait
end

2. Generating and Displaying the Captcha Code

when this sprite clicked
ask [Enter your Scratch username:] and wait
set [username v] to (answer)
set [server_response v] to (brackets [get_captcha] (username) :: extension)
if <(server_response) contains [|]> then
  set [idx v] to [1]
  set [rule_text v] to []
  repeat until <(letter (idx) of (server_response)) = [|]>
    set [rule_text v] to (join (rule_text) (letter (idx) of (server_response)))
    change [idx v] by (1)
  end
  change [idx v] by (1)
  set [verification_code v] to []
  repeat until <(idx) > (length of (server_response))>
    set [verification_code v] to (join (verification_code) (letter (idx) of (server_response)))
    change [idx v] by (1)
  end
  say (rule_text) for (5) secs
  say (join [Your code is: ] (verification_code))
else
  say [Server error. Please try again.] for (3) secs
end
